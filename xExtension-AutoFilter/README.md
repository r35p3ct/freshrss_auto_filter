# FreshRSS Auto Filter Extension

Расширение для автоматического определения и фильтрации рекламных записей в FreshRSS с использованием AI (OpenRouter API).

## Возможности

- **Автоматическое определение рекламы** с помощью AI (OpenRouter API)
- **Гибкая система меток**:
  - `Реклама` — высокая уверенность (>80%)
  - `Возможно реклама` — средняя уверенность (>50%)
- **Настраиваемые действия**:
  - Скрытие рекламы (пометка прочитанным)
  - Опускание подозрительных записей в ленте
  - Только добавление меток
- **Проверка при получении** новых записей
- **Ручная проверка** отдельных записей
- **Массовая проверка** выбранных записей

## Установка

1. Скопируйте папку `xExtension-AutoFilter` в директорию расширений FreshRSS:
   ```
   ./extensions/AutoFilter/
   ```

2. Включите расширение в настройках FreshRSS:
   - Перейдите в `Настройки` → `Расширения`
   - Найдите `Auto Filter` и нажмите `Включить`

3. Настройте расширение:
   - Перейдите в `Настройки` → `Расширения` → `Auto Filter` → `Настройки`
   - Укажите ваш API ключ OpenRouter
   - Выберите модель (например, `openai/gpt-3.5-turbo`)
   - Настройте пороги уверенности и действия

## Настройка

### API ключ

Получите API ключ на [openrouter.ai/keys](https://openrouter.ai/keys)

### Модель

Рекомендуемые модели:
- `openai/gpt-3.5-turbo` — быстро и недорого
- `openai/gpt-4` — более точное определение
- `anthropic/claude-3-haiku` — хорошая альтернатива

### Пороги уверенности

- **Порог для рекламы** (по умолчанию 0.8) — при уверенности выше этого значения запись считается рекламой
- **Порог для возможной рекламы** (по умолчанию 0.5) — при уверенности выше этого значения запись помечается как возможная реклама

### Действия

#### Для рекламы:
- **Ничего не делать** — только добавить метку
- **Скрыть** — пометить прочитанным и скрыть из ленты
- **Пометить прочитанным** — не скрывать, но пометить как прочитанное
- **Только метка** — добавить метку без дополнительных действий

#### Для возможной рекламы:
- **Ничего не делать** — только добавить метку
- **Опустить в ленте** — сдвинуть дату записи на 1 день назад
- **Только метка** — добавить метку без дополнительных действий

## Использование

### Автоматическая проверка

Если включена опция "Проверять при получении", расширение будет автоматически проверять новые записи при их получении.

### Ручная проверка

1. Найдите кнопку 🛡️ в заголовке записи
2. Нажмите для проверки записи на рекламу
3. Запись будет помечена соответствующей меткой

### Массовая проверка

1. Выберите несколько записей
2. Нажмите кнопку "🛡️ Проверить выбранные"
3. Дождитесь завершения проверки

## Фильтрация по меткам

Используйте встроенные средства FreshRSS для фильтрации:

1. Создайте ярлык с фильтром по метке:
   - `label:advertisement` — показать только рекламу
   - `-label:advertisement` — скрыть рекламу
   - `label:possible_advertisement` — показать возможные рекламы

2. Используйте ярлыки для быстрого доступа к отфильтрованным записям

## Визуальные индикаторы

- **Красная рамка** — запись определена как реклама
- **Оранжевая рамка** — запись возможна как реклама
- **Фон записи** — слегка окрашен в соответствующий цвет

## API

### Проверка одной записи

```
GET /api/p.php?c=autoFilter_openrouter&a=checkEntry&entry_id={ID}
```

Ответ:
```json
{
    "success": true,
    "entry_id": 123,
    "analysis": {
        "is_advertisement": true,
        "confidence": 0.95,
        "reason": "Коммерческое продвижение товара",
        "label": "advertisement"
    }
}
```

### Массовая проверка

```
POST /api/p.php?c=autoFilter_openrouter&a=checkBatch
{
    "entry_ids": [123, 456, 789]
}
```

## Требования

- FreshRSS 1.20 или выше
- PHP 7.4 или выше
- API ключ OpenRouter
